allows free commercial and personal use with no attribution required.
Do NOT mix in assets from Icons8 or other attribution/paid-tier sites here
(see project licensing decision — lucide-react for icons, unDraw for scenes).

Each file was downloaded as-is from `cdn.undraw.co` and recolored in place:
the stock unDraw accent `#6c63ff` → OrbiSave green `#00ab00`. No other edits.

`AppStateIcon` of each state in `lib/app-states.ts`) and directly by the
